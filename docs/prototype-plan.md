# **Rencana Prototipe Nusantara**

**Prototipe Ke-1:**
- **Fokus:** Membuat **Nusantara Interpreter** menggunakan C++ dan ANTLR4.
- **Tujuan:** Interpreter pertama yang akan digunakan untuk membangun interpreter Nusantara menggunakan bahasa Nusantara itu sendiri pada prototipe berikutnya.

**Prototipe Ke-2:**
- **Fokus:** Membuat **Nusantara Interpreter** menggunakan bahasa Nusantara.
- **Tujuan:** Menggunakan interpreter yang sudah dibuat untuk membangun **Nusantara Just-In-Time (JIT)** pada prototipe berikutnya.

**Prototipe Ke-3:**
- **Fokus:** Mengimplementasikan **Nusantara Just-In-Time (JIT)**.
- **Tujuan:** Menggunakan **Nusantara JIT** untuk membangun **Nusantara Code Generation (C++)** pada prototipe berikutnya.

**Prototipe Ke-4:**
- **Fokus:** Implementasi **Nusantara Code Generation** untuk menghasilkan kode **C++**.
- **Tujuan:** Menggunakan code generator ini untuk menghasilkan file executable, yang kemudian digunakan untuk membangun **Nusantara Code Generation (C)** pada prototipe berikutnya.

**Prototipe Ke-5:**
- **Fokus:** Mengembangkan **Nusantara Code Generation** untuk **C**.
- **Tujuan:** Menggunakan code generator **C** ini untuk menghasilkan file executable, yang kemudian digunakan untuk membangun **Nusantara Code Generation (Assembly)** pada prototipe berikutnya.

**Prototipe Ke-6:**
- **Fokus:** Membuat **Nusantara Code Generation** untuk **Assembly**.
- **Tujuan:** Menggunakan **Nusantara Code Generation (Assembly)** untuk menghasilkan file executable, yang kemudian digunakan untuk membangun **Nusantara Assembler** pada prototipe berikutnya.

**Prototipe Ke-7:**
- **Fokus:** Mengimplementasikan **Nusantara Assembler**.
- **Tujuan:** Menggunakan file objek dari **Nusantara Assembler** dan melakukan linking untuk menghasilkan file executable, yang kemudian digunakan untuk membangun **Nusantara Code Generation Machine Code** pada prototipe berikutnya.

**Prototipe Ke-8:**
- **Fokus:** Membuat **Nusantara Code Generation Machine Code**.
- **Tujuan:** Menggunakan **Nusantara Code Generation Machine Code** untuk menghasilkan file executable, yang kemudian digunakan untuk membangun **Nusantara Linker** pada prototipe berikutnya.

**Prototipe Ke-9:**
- **Fokus:** Mengembangkan **Nusantara Linker**.
- **Tujuan:** Menggunakan **Nusantara Linker** untuk membangun Nusantara yang sepenuhnya fungsional dengan kemampuan untuk menghasilkan executable.

---

**Catatan:**  
Setiap prototipe dibuat dengan menggunakan prototipe sebelumnya untuk membangun fitur berikutnya. Setiap langkah bertujuan untuk meningkatkan kemampuan bahasa Nusantara hingga mencapai tahap di mana Nusantara bisa mengkompilasi dirinya sendiri dengan **Linker** dan menghasilkan **file executable**.
