# Tujuan Bahasa Pemrograman Nusantara

Membuat sebuah bahasa pemrograman adalah tujuan yang ambisius dan membutuhkan banyak usaha. Itulah mengapa saya menjadikan proyek ini sebagai proyek sampingan. Jadi, mengapa saya ingin mempersulit diri dengan membuat bahasa ini?

Proyek ini akan menjadi perjalanan saya untuk menjawab semua pertanyaan saya tentang bahasa pemrograman, seperti apa itu bahasa pemrograman, mengapa dibuat, bagaimana cara membuatnya, bagaimana cara kerjanya, dan lain sebagainya.

Jadi, saat ini belum ada tujuan spesifik untuk bahasa pemrograman ini.
